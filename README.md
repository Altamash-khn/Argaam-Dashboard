# Argaam Dashboard Clone

This project is a **frontend dashboard task** inspired by the original [Argaam](https://www.argaam.com/en) website.  
It replicates the dashboard’s design and layout using **React** and **TailwindCSS**, focusing on clean UI and modular component structure.

---

## Live Demo

**URL:** [https://argaam-dashboard-blue.vercel.app/](https://argaam-dashboard-blue.vercel.app/)

---

## Tech Stack

- **React 19** – Component-based UI development
- **TailwindCSS 4** – Utility-first styling framework
- **Lucide React** – Modern icon library
- **Recharts** – Data visualization and charts
- **clsx / class-variance-authority / tailwind-merge** – Conditional and optimized class handling
- **Vite** – Fast build tool and development server

---

## Features

- Modern **dashboard layout** inspired by Argaam
- **Responsive** and clean user interface
- **Reusable components** with semantic structure
- **Interactive charts** powered by Recharts
- Built with **Vite** for fast builds and hot reload

## Installation & Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/Altamash-khn/Argaam-Dashboard.git
   cd Argaam-Dashboard
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run the development server**

   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```
